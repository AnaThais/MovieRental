Desafio:

Você foi contratado para desenvolver um sistema que permita o controle de aluguéis de filmes em uma locadora.
Utilizando qualquer linguagem de programação, esperamos que você desenvolva um sistema web simples que possua as seguintes funcionalidades:

1) Cadastro de filmes, contendo pelo menos o título, ano, diretor e gênero.
2) Listagem dos filmes por gênero.
3) Cadastro de clientes, contendo pelo menos o nome e email do cliente.
4) Registro de locação, relacionando o filme a um cliente em um período (início, término)
5) Listagem de filmes locados.


- Você pode utilizar qualquer linguagem e/ou framework de programação. Desenvolver em Python e Django será um diferencial.
- Qualquer banco de dados pode ser utilizado para armazenar as informações, inclusive SQLite.
- Não há necessidade de escrever testes, embora seja considerado um diferencial.
- Não há necessidade de desenvolver/utilizar um layout (bootstrap, css, etc), embora seja considerado um diferencial.
- Você pode optar por desenvolver as funcionalidades como API também, ao invés do tradicional "CRUD" web.
- Utilize um repositório (github, gitlab) para publicar seu código e nos enviar.
